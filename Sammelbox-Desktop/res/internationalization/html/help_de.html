    Was ist Sammelbox?
    [Go to top]

    Sammelbox ist ein offenes open-source Managementsystem für Sammlungen. Sammelbox erlaubt es maßgeschneiderte Sammlungen (genannt "Alben") mit verschiedenen Feldtypen zu erstellen. Sammelbox unterstützt unter anderem folgende Feldtypen: Text, Bewerungen, Ganzzahlen, Dezimalzahlen, urls(Web-Links), Bilder und mehr...

    Zusätzlich unterstützt Sammelbox das Import- und Exportieren der Alben auf mobile Geräte, welche mittels Sammelbox-Android-Applikation aufrufbar sind. Verschieden Albenansichten sowie Suchfunktionen erlauben ihnen ein einfaches Verwalten ihrer Sammlungen!

    Besuchen sie bitte www.Sammelbox.org für weitere Informationen

    Einige nützliche Funktionen fehlen! Was kann ich tun?
    [Go to top]

    Sammelbox wurde für ein intuitives und einfaches Arbeiten entwickelt. Nur die wichtigsten Funktionen sind auf den ersten Blick sichtbar. So sind viele Funktionen in den Pop-up Menus versteckt. Zum Beisspiel gibt ihnen ein Rechtsklick auf die Albumliste die Möglichkeit, Alben zu sortieren. Sollten sie einige Funktionen denoch nicht finden, so besuchen sie bitte: www.sammelbox.org oder informieren uns per E-Mail: contact@sammelbox.org

    Wie kann ich schnellstmöglich viele Einträge hinzufügen?
    [Go to top]

    Der schnellste Weg viele Einträge auf einnmal zu erstellen führt über Tabllenansicht. Die Tabllenansicht erlaubt es Ihnen wie in Microsoft Excel oder Open Office Calc, mehrere Einträge übersichtlich mittels einer Tabelle einzufügen. Beachten Sie: es ist nicht möglich Bilder in der Tabellenansicht hinzuzufügen! Diese können Sie jedoch nachträglich einfach hinzufügen.

    Was ist ein gespeicherter Filter?
    [Go to top]

    Sammelbox ermöglicht Ihnen das Ausführen von komplexen Abfragen. Um das Wiederholen von jeweils neu definierten Abfragen (Suchen) zu vermeiden, können sie diese unter einem passendem Namen abspeichern. Gespeicherte Abfragen (Filter) können dabei weiterhin editiert werden. Beachten sie: falls sie die Albumstruktur bearbeiten (z.B. durch löschen eines Feldes), so wird ihre Suche möglicherweise nicht mehr funktionieren. In diesem Fall müssen sie die Suche anpassen!

    Wie kann ich meine Alben sortieren?
    [Go to top]

    Automatisch wird Ihr Album(Sammlung) nach dem ersten Feld sortiert. Dennoch haben sie die Möglichkeit ihr Album nach jedem Feld zu sortierten. (Rechtsklick auf die Albumliste und ein Menu öffnet sich) Weiterhin können sie komplett auf das Sortieren verzichten. In diesem Fall kommen die Einträge im Album in der Reihenfolge, in welcher sie hinzugefügt wurden.
