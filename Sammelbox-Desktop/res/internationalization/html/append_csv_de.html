Anhängen von CSV Datensätzen

Sammelbox erlaubt das importieren von CSV (comma seperated value) Dateien in Form neuer Alben. Solche CSV Dateien können mithilfe von Tabellenkalkulationen wie beispielsweise Microsoft Excel oder Open Office Calc erstellt werden. Der Inhalt solcher Dateien ist dabei tabular wobei jede Kolone durch einen Komma oder ein anderes Symbol getrennt ist.
Achtung! : Um CSV Einträge zu einem bestehenden Album hinzufügen zu können müssen alle Kolonen dem jeweiligen Album Feldtypen entsprechen.

Beispiel:

Book Titel;Autor;Seiten
Book 1;Author 1;42
Book 2;Author 2;21
Book 3;Author 3;88

Bilder aus Kolone importieren

Sammelbox erlaubt das importieren von Bildern für jeden CSV Eintrag. Die Bilder müssen hierbei durch ihren kompletten Pfad definiert werden. (z.B. C:/Meine-Bilder/Bild1.jpg) Falls mehrere Bilder für einen Eintrag importiert werden sollen, so ist es notwendig alle Bildpfade durch ein Trennzeichen zu trennen.

Feld-Typen definieren

Sammelbox importiert standardmässig alle Kolonen (Felder) als einen Text. Es ist jedoch möglich Feldtypen manuell zu definieren. Hierzu muss der Typ mittels '~' an den Kolonen-Namen angehängt werden.

Example:

Buch Titel~TEXT;Author~TEXT;Pages~INTEGER

Verfügbare Feldtypen und Werte

Die folgende Tabelle beinhaltet alle verfügbaren Feld typen und aktzeptieren Werte:

Typ Aktzeptiere Werte
TEXT Fast alles kann als Text interpretiert werden. Bitte beachten Sie jedoch dass dies ihre Suchmöglichkeiten einschränkt.
DECIMAL Dezimalzahlen (z.B: 42.21)
INTEGER Ganzzahlen (z.B: 21)
DATE A date in the format specified by the settings. (E.g: 16/01/1988) Please verify your settings if you are unsure about the current format!
URL A web link (E.g: www.sammelbox.org)
OPTION A yes, no & unknown field. Accepted values are 'YES', 'NO' and 'UNKNOWN'
STAR_RATING A star rating between 0 and 5. Accepted values are 'ZERO_STARS', 'ONE_STAR', 'TWO_STARS' 'THREE_STARS', 'FOUR_STARS' and 'FIVE_STARS'.
